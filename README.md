## 📂 Featured Projects

### 🛡️ [Defence Intelligence Toolkit](https://github.com/AntoineGuigui/defence-intelligence-toolkit)

* **Impact:** Actively used by MBDA's industrial strategy team for COMEX briefings covering 84+ companies across 9 countries.
* `Flask` `NLP` `python-pptx` `XML`

### 🔍 [Company Intelligence Generator](https://github.com/AntoineGuigui/company-intelligence)
An automated OSINT data pipeline that orchestrates DuckDuckGo web scraping, Yahoo Finance API data fetching, and GPT-4o structured extraction to feed a centralized Excel database. 
* **Impact:** Drastically reduced manual research time; outputs directly power the Defence Intelligence Toolkit.
* `Streamlit` `GPT-4o` `Web Scraping` `APIs`

### 📊 [Store Clustering & Assortment Strategy](https://github.com/AntoineGuigui/store-clustering)
Clustering of ~200 retail stores using dimensionality reduction and unsupervised learning to optimize product assortment by store profile. Identified 3 actionable clusters and built a data-driven commercial strategy (core vs. differentiated vs. excluded references).
* `Python` `scikit-learn` `K-Means` `PCA`

### 📈 [Predicting Student Performance](https://github.com/AntoineGuigui/student-performance-prediction)
An ML project exploring socio-economic factors in academic success. Benchmarked 9 regression models and tackled severe class imbalance using SMOTE — shifting a misleading 88.5% baseline accuracy into a high-utility early-warning tool.
